🔹 Day 1: Core Syntax & Control Flow
Goal: Be comfortable writing small programs with input, output, and decisions.

- Topics:
- Java program structure (class, main).
- Data types (int, double, char, boolean, String).
- Operators (+, -, *, /, %, relational, logical).
- Conditionals (if-else, switch).
- Loops (for, while, do-while).

- Practice:
- Calculator.
- Leap year checker.
- Multiplication table generator.
- Star pattern printing.

🔹 Day 2: Arrays, Strings & Methods
Goal: Handle collections of data and modularize code.

- Topics:
- Arrays (1D, traversal, min/max, reverse).
- Strings (length, charAt, substring, equals, toCharArray).
- Methods (parameters, return values, static methods).
- Recursion basics.

- Practice:
- Reverse an array.
- Palindrome string checker.
- Count vowels/consonants in a string.
- Factorial (iterative + recursive).
- Fibonacci series (recursive).

🔹 Day 3: Object-Oriented Basics & Collections
Goal: Understand OOP enough to structure DSA problems cleanly.

- Topics:
- Classes & objects.
- Constructors.
- Encapsulation (fields + getters/setters).
- Basic Collections (ArrayList, HashMap).
- Exception handling (try-catch).

- Practice:
- Student class → calculate grade.
- BankAccount class → deposit/withdraw.
- Store names in ArrayList and print.
- Store student marks in HashMap.
- Divide numbers with exception handling.
